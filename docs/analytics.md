# Analytics

Tablofy provides two ways to understand your data: **rule-based insights** and **statistical helpers**.

## Insights

Call `data.insights()` to get a list of human-readable observations about your dataset.

These are **deterministic** — entirely rule-based, not generated by AI or an LLM.

```python
for note in data.insights():
    print(f"  - {note}")
```

Example output:

```
  - Dataset has 1000 rows and 8 columns.
  - No missing values detected.
  - No duplicate rows found.
  - Strong correlations detected (1): ad_spend — sales (r = 0.89).
  - Column 'id' is a high-cardinality, unique-like column.
  - Low-cardinality categorical columns (1): ['region'].
  - sales: min = 10, max = 500.
  - profit: min = 2, max = 120.
  - sales: 12 potential outlier(s) (1.2% of rows).
```

### What insights detect

| Detection | Description |
|-----------|-------------|
| Overview | Row and column counts |
| Missing values | Count and percentage of empty cells |
| Duplicates | Count and percentage of duplicate rows |
| Strong correlations | Pairs with \|r\| ≥ 0.7 |
| High-cardinality | Columns where every value is unique |
| Low-cardinality | Categorical columns with few unique values |
| Numeric range | Min and max per numeric column |
| Outlier hints | IQR-based outlier counts per numeric column |

## Statistics

Access statistics via the `data.stats` object.

### describe()

Descriptive statistics for all columns in one call:

```python
data.stats.describe()
```

### correlation()

Correlation matrix of numeric columns:

```python
data.stats.correlation()
```

Values range from -1 (perfect negative) to +1 (perfect positive). 0 means no linear relationship.

### covariance()

Covariance matrix of numeric columns:

```python
data.stats.covariance()
```

### outliers()

Detect outliers using the IQR (interquartile range) method.

```python
# Returns a DataFrame of outlier rows (with a _outlier_distance column)
data.stats.outliers("sales")

# Returns a dict with count and row indices
data.stats.outliers("sales", return_indices=True)

# Custom IQR multiplier (default 1.5, higher = fewer outliers)
data.stats.outliers("sales", multiplier=2.0)
```

The IQR method flags values below `Q1 - 1.5×IQR` or above `Q3 + 1.5×IQR` as potential outliers.

### Univariate statistics

```python
data.stats.mean("sales")          # mean (average)
data.stats.median("sales")        # median (middle value)
data.stats.std("sales")           # standard deviation
data.stats.min("sales")           # minimum value
data.stats.max("sales")           # maximum value
data.stats.quantile("sales", 0.5) # any quantile
data.stats.value_counts("region") # frequency counts
```

All univariate methods validate that the column exists (and is numeric where applicable).
